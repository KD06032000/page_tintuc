Development

Cài đặt môi trường

$ git clone git@gitlab.com:apecsoftasia/cms-core.git /path/to/proj_dir

Với https

$ git clone https://gitlab.com/apecsoftasia/cms-core.git /path/to/proj_dir

$ cd /path/to/proj_dir

$ git checkout dev && git pull

$ composer install

cài đặt website

  • import file db trong folder db thư mục ngoài cùng.
  • Tạo file config.php thư mục ngoài cùng. Copy file config_simple vào file config.php và cài đặt thông tin để connect database
  • tạo folder config cùng thư mục file config.php và tạo file settings.cfg (file này sữ lưu thông tin cấu hình chung của website)
  • Tạo folder media trong public

Thông tin đăng nhập admin

url: domain.com/admin

Tài khoản đăng nhập:
    user: admin
    Password: ssc@2021

Quản lý file với Laravel Mix

#####Tài liệu tham khảo: https://laravel-mix.com/

  • Cài đặt npm để quản lý thư viện js

  • command 

    cd /path/to/proj_dir
npm init

    Lưu ý: Khi thực hiện các bước yêu cầu thì enter không sửa gì thêm tránh conflig

  • Config thêm, bớt file css,js

Để thêm bớt được file js,css thì ta sẽ thêm file vào hàm trong file webpack.mix.js

Với js

mix.combine(input,output) (trong đó: Input là các file js cần mix có thể 1 hoặc nhiều file, Output là file được mix từ các file thành 1)

Ví Dụ:

mix.combine([
  'public/js/jquery.js',
  'public/js/custom.js',
], 'public/js/app.js');

Với css

Các tham số cũng giống như thêm js mix.styles(input,output)

mix.styles(
  [
    'public/plugins/toastr/toastr.min.css',
    'public/plugins/select2/css/select2.min.css',
    'public/fonts/fontawesome-pro-5.8.2-web/css/all.min.css',
    'public/fonts/elegantIcon/elegantIcon.css',
    'public/css/bootstrap.min.css',
    'public/css/slick.css',
    'public/css/main.css',
    'public/css/custom.css',
  ],
  'public/css/app.css')
  • Lưu ý: Khi thêm 1 fonts hay bất kỳ file css mà load đường dẫn 1 font, ảnh ở folder # không cùng cấp với folder css thì cần thêm code copy để khi mix css lại không bị lỗi font, ảnh
    mix.copyDirectory('public/fonts/fontawesome-pro-5.8.2-web/webfonts/', 'public/webfonts/');

Chạy wabpack:

  • Build file trên môi trường development
    npm run dev
  • build file trên môi trường production
    npm run prod
  • Auto build mỗi khi mình sửa file css,js thuộc những file mình import 
    npm run watch
